Udacity HTML Basics

Basic HTML document structure:

  1. Doctype declaration informs browser what kind of document it's rendering
  2. HTML tags surround document
  3. Head contains metadata, title, links to CSS and Javascript
  4. Body contains renderable contents

Basic HTML Elements

NameTagPurpose/function
Emphasis<em>content</em>Creates italic text
Bold<b>content</b>Creates bold text
Anchor<a>content</a>Creates a hyperlink
Image<img src="#" alt="#">Creates an image. This tag is self-closing
Line Breakcontent<br>contentCreates a line break. This is an in-line element and self-closing
Paragraph<p>content</p>Creates a paragraph and line-breaks. This is a block element
Division<div>content</div>Creates a container for text and other elements. This is a block element
Span<span>content</span>Creates an in-line container for text

Notes of Interest

  • White space is mostly ignored by HTML, except inside tags
  • Inline elements don't create line breaks or box (block), while block elements do
